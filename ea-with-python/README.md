## PSO

> PSO, particle swarm optimization

### 基础版本

1. 初始化粒子群，包括位置、速度
2. 计算每个粒子的适应度，初始化个体极值与全局极值
3. 遍历每个粒子，做更新操作
4. 计算新的适应度，当高于个体极值或高于全局极值时，更新相关变量
5. 到达结束条件时，结束程序
